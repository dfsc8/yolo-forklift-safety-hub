功能说明 / Function Description：

  • 错误计数： 仅在设备发生从“正常”变为“报警”的瞬间增加。/ Error count increases only when device status changes from normal to alarm.
  • 运行时间： 由前端根据服务器下发的启动时间戳每秒动态计算，显示为 HH:MM:SS。/ Runtime is calculated dynamically every second based on start timestamp from server (HH:MM:SS).
  • 离线检测： 10 秒无数据自动判定，离线后停止运行时间计算。/ Offline is judged after 10s no data, runtime calculation stops when offline.
